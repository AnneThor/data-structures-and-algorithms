Class: BinarySearchTree

BinarySearchTree()

The BinarySearchTree class creates binary trees that are composed of TreeNodes and includes methods to add, delete and traverse in various orders (pre, in, post). The generic BinaryTree class fills the tree using a value sort method, saving to a left leaf if the value is lower and to the right leaf if value is greater. The generic BST does not do any self balancing.
