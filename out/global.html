leftJoin(left, right) → {object}

Left join accepts two hash tables as arguments and returns a left join (i.e. all values in the left table are returned with the values from the right table appended to the result row if they exist)
Parameters:
Name Type Description
left object a hashtable that has strings as keys and a synonym of the keys for values, so key -> value --> string: string
right object a hashtable that has strings for keys and antonyms for values, so key:value -> string: string

- an object that contains objects with they key values from the left join and the synonym (and antonym if applicable else null) i.e. { {key: string, syn: sting, ant: string}, { key: string...}}
